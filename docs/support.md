# Support

← [Übersicht / Overview](index.md)

## Deutsch

Bei Fragen, Fehlerberichten oder Verbesserungsvorschlägen wende dich bitte an:

- **E-Mail:** [mail@jch-technologies.de](mailto:mail@jch-technologies.de)
- **Telefon:** [+49 15679 698581](tel:+4915679698581)

Bei Verbindungsproblemen bitte folgende Infos in der E-Mail mitschicken:

- Betroffene Funktion (MMS-Client / Server-Simulator / GOOSE / SCL-Editor)
- IED-Hersteller + Modell (sofern teilbar)
- macOS-/iPadOS-Version + App-Version (im About-Screen)
- Was du erwartet hast vs. was passiert ist
- Idealerweise: Auszug aus dem Log-Tab + ggf. anonymisierte SCD-Datei

## English

For questions, bug reports or feature requests, please reach out to:

- **E-mail:** [mail@jch-technologies.de](mailto:mail@jch-technologies.de)
- **Phone:** [+49 15679 698581](tel:+4915679698581)

For connection problems, please include in your e-mail:

- Affected feature (MMS client / Server simulator / GOOSE / SCL editor)
- IED vendor and model (where it can be shared)
- macOS / iPadOS version + app version (shown in the About screen)
- Expected behaviour vs. what happened
- Ideally: an excerpt from the Log tab and an anonymised SCD file
